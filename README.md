## 🌷 About Me

안녕하세요.  
저는 **품질 검증**, **자동화 테스트**, **프로세스 개선**에 집중하는 **QA / SDET**입니다.

현재는 **AhnLab Blockchain Company**에서  
**Klip Wallet / ABC WaaS / Web · Mobile 서비스 품질 검증**을 중심으로 일하고 있습니다.

사용자가 실제로 겪는 흐름을 기준으로 테스트하고,  
반복 업무는 자동화하며,  
품질을 더 안정적으로 만들기 위한 구조와 기준까지 함께 고민합니다.
